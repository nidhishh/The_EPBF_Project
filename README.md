# Electron Beam Wire Deposition Simulation

## Project Overview
The Electron Beam Wire Deposition (EBWD) process is a novel manufacturing technique that utilizes an electron beam to deposit material in a wire format. This project aims to simulate the EBWD process for better understanding and optimization of parameters involved in additive manufacturing.

## Physics Model
The simulation is based on the fundamental principles of electron beam physics, including the interaction of high-energy electrons with materials, heat transfer, and phase change dynamics. The key components of the physics model include:
- **Electron Trajectory Calculation:** Determining the path of electrons based on their initial energy and material properties.
- **Heat Generation:** Modeling heat produced during the material deposition process and its impact on the substrate.
- **Material Behavior:** Understanding how materials change states under varying temperature and pressure conditions.

## Current Implementation
The current version implements the basic functionality for simulating the deposition of material using an electron beam. Key features include:
- Basic GUI for input parameter specification
- 2D visualization of the deposition process
- Export capabilities for result analysis

## Future Work
Looking ahead, the following enhancements are planned:
- Implementation of 3D visualization capabilities
- Incorporation of advanced material models, including alloy behavior and impurity effects.
- Optimization algorithms for parameter tuning in the deposition process.
